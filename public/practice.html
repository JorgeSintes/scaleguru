Practice

Practice mode

The practice mode is a functional tool designed for musicians to refine their mastery of scales and keys.

Select the keys and scales you want to practice and start playing! The tool will ask for scales in different keys and you will have to play them. Choose from different practice routines:

  • Mantain scale, change key: This mode will ask for all keys before switching to a new scale. Ideal to familiarize yourself with the sound of a specific scale.
  • Mantain key, change scale: This mode will ask for all scales before switching to a new key.
  • Random scale: The true challenge. This mode will ask for a random scale in a random key every time.

Struggling with a specific scale? Switch to Inspector Mode to practice more slowly!
